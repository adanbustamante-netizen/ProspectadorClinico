
# Prospector Hospitalario Internacional

Aplicación web desarrollada con:

- Streamlit
- Gemini API
- Python

## Funciones

- Prospectar hospitales internacionales
- Identificar compatibilidad con universidades online
- Clasificar instituciones por capas estratégicas
- Generar rankings ejecutivos
- Detectar riesgos regulatorios
- Descargar reportes en Markdown

## Instalación

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Configuración de Gemini

Crear:

`.streamlit/secrets.toml`

```toml
GEMINI_API_KEY = "TU_API_KEY"
```

## Deploy

Compatible con:

- GitHub
